## 三、中间件部署

### 3.1 目录准备

```bash
sudo mkdir -p /data/unimarket/{rocketmq/{namesrv/logs,broker/{logs,store,conf}},elasticsearch/data,xxl-job/logs,mysql,redis}
sudo chmod -R 777 /data/unimarket/
```

### 3.2 内核参数调优

```bash
echo "vm.max_map_count=262144" | sudo tee -a /etc/sysctl.conf
sudo sysctl -p
```

### 3.3 Docker 镜像加速器（可选）

```bash
sudo mkdir -p /etc/docker
sudo tee /etc/docker/daemon.json <<-'EOF'
{
  "registry-mirrors": ["https://fq0nnhbl.mirror.aliyuncs.com"]
}
EOF
sudo systemctl daemon-reload
sudo systemctl restart docker
```

### 3.4 docker-compose.yml

文件路径：`/data/unimarket/docker-compose.yml`

```yaml
version: '3.8'

services:
  # RocketMQ NameServer
  rmqnamesrv:
    image: apache/rocketmq:5.1.4
    container_name: rmqnamesrv
    ports:
      - "9876:9876"
    environment:
      - JAVA_OPT_EXT=-Xms128m -Xmx128m -Xmn64m
    volumes:
      - /data/unimarket/rocketmq/namesrv/logs:/home/rocketmq/logs
    command: sh mqnamesrv
    restart: always

  # RocketMQ Broker
  rmqbroker:
    image: apache/rocketmq:5.1.4
    container_name: rmqbroker
    ports:
      - "10911:10911"
      - "10909:10909"
    environment:
      - JAVA_OPT_EXT=-Xms256m -Xmx256m -Xmn64m
    volumes:
      - /data/unimarket/rocketmq/broker/conf/broker.conf:/home/rocketmq/rocketmq-5.1.4/conf/broker.conf:ro
      - /data/unimarket/rocketmq/broker/logs:/home/rocketmq/logs
      - /data/unimarketroker/store:/home/rocketmq/store
    command: sh mqbroker -n rmq/rocketmq/bnamesrv:9876 -c /home/rocketmq/rocketmq-5.1.4/conf/broker.conf
    depends_on:
      - rmqnamesrv
    restart: always

  # Elasticsearch 8.11.0 + IK + Pinyin
  elasticsearch:
    image: docker.elastic.co/elasticsearch/elasticsearch:8.11.0
    container_name: elasticsearch
    environment:
      - discovery.type=single-node
      - xpack.security.enabled=false
      - ES_JAVA_OPTS=-Xms256m -Xmx256m
      - indices.memory.index_buffer_size=10%
    volumes:
      - /data/unimarket/elasticsearch/data:/usr/share/elasticsearch/data
    ports:
      - "9200:9200"
    restart: always

  # XXL-Job Admin
  xxl-job-admin:
    image: xuxueli/xxl-job-admin:2.4.0
    container_name: xxl-job-admin
    environment:
      - JAVA_OPTS=-Xms128m -Xmx256m
      - PARAMS=--spring.datasource.url=jdbc:mysql://host.docker.internal:3306/xxl_job?useUnicode=true&characterEncoding=UTF-8&autoReconnect=true&serverTimezone=Asia/Shanghai \
               --spring.datasource.username=root \
               --spring.datasource.password=123456
    extra_hosts:
      - "host.docker.internal:host-gateway"
    ports:
      - "8088:8080"
    restart: always
```

### 3.5 RocketMQ Broker 配置文件

文件路径：`/data/unimarket/rocketmq/broker/conf/broker.conf`

```properties
brokerClusterName = DefaultCluster
brokerName = broker-a
brokerId = 0
deleteWhen = 04
fileReservedTime = 48
brokerRole = ASYNC_MASTER
flushDiskType = ASYNC_FLUSH
brokerIP1=你的服务器IP
brokerIP2=你的服务器IP
listenPort=10911
namesrvAddr=rmqnamesrv:9876

# 最大延迟时间（10天 = 864000秒）
timerMaxDelaySec = 864000

# 内存优化配置（适合小数据量）
sendMessageThreadPoolNums = 2
pullMessageThreadPoolNums = 2
adminBrokerThreadPoolNums = 2
flushConsumerOffsetThreadPoolNums = 1

# 降低文件大小和数量
mapedFileSizeCommitLog = 67108864
mapedFileSizeConsumeQueue = 300000

# 降低刷盘间隔
flushIntervalCommitLog = 500
flushIntervalConsumeQueue = 1000

# 降低消费队列数量
defaultTopicQueueNums = 2
```

**注意**：将 `brokerIP1` 和 `brokerIP2` 替换为实际服务器 IP。

### 3.6 启动中间件

```bash
cd /data/unimarket
docker-compose up -d
```

### 3.7 安装 Elasticsearch 插件

```bash
# 进入容器安装 IK 中文分词器
docker exec elasticsearch bin/elasticsearch-plugin install \
  https://release.infinilabs.com/analysis-ik/stable/elasticsearch-analysis-ik-8.11.0.zip \
  --batch

# 安装 Pinyin 分词器
docker exec elasticsearch bin/elasticsearch-plugin install \
  https://release.infinilabs.com/analysis-pinyin/stable/elasticsearch-analysis-pinyin-8.11.0.zip \
  --batch

# 重启 ES 使插件生效
docker restart elasticsearch
```

## 六、应用部署（Java 后端）

### 6.1 项目结构

```
UniMarket-backend/
├── unimarket-web/          # Web 层
├── unimarket-core/         # 核心业务
├── unimarket-ai/           # AI 功能
├── unimarket-admin/        # 后台管理
├── unimarket-security/     # 安全模块
├── unimarket-search/       # 搜索功能
├── unimmarket-recommend/   # 推荐功能
```

### 6.2 配置文件修改

修改各模块的 `application.yml` 中的连接地址：

```yaml
# Redis 配置
spring:
  data:
    redis:
      host: 服务器IP
      port: 6379

# MySQL 配置
  datasource:
    url: jdbc:mysql://服务器IP:3306/unimarket?useUnicode=true&characterEncoding=UTF-8
    username: root
    password: 你的密码

# RocketMQ 配置
rocketmq:
  name-server: 服务器IP:9876
  producer:
    group: unimarket-producer-group

# Elasticsearch 配置
spring:
  elasticsearch:
    uris: http://服务器IP:9200

# XXL-Job 配置
xxl:
  job:
    admin-addresses: http://服务器IP:8088/xxl-job-admin
    executor:
      appname: unimarket-executor
      port: 9999
```

### 6.3 构建与启动

```bash
# 构建项目
mvn clean package -DskipTests

# 启动应用（使用 IDE 或命令行）
java -jar unimarket-web/target/unimarket-web.jar
```

或使用 IDE（如 IntelliJ IDEA）直接运行 `UniMarketApplication.java`

## 九、注意事项

1. **服务器 IP**：所有配置中的 `服务器IP` 需要替换为实际 IP 地址
2. **数据库密码**：MySQL 和 Redis 的密码需要根据实际情况修改
3. **防火墙**：确保上述端口在防火墙中开放
4. **时间同步**：确保服务器与应用服务器时间同步（NTP），避免 MQ 延迟消息问题
5. **数据持久化**：所有数据目录都已挂载到宿主机的 `/data/unimarket/`
